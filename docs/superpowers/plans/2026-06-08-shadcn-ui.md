# shadcn/ui (Token-Themed) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Install and configure shadcn/ui on the existing Next.js 16 / React 19 / Tailwind v4 stack, themed with a token-based World Cup palette and light/dark toggle, and prove it works via a rebuilt demo home page.

**Architecture:** shadcn/ui's CLI copies component source into the repo (`components/ui/`). Theming is token-based: raw `oklch()` values live as CSS variables in `app/globals.css` (`:root` + `.dark`), mapped to utilities via Tailwind v4's `@theme inline`; components consume only semantic tokens. Dark mode is class-based via `next-themes`. The home page becomes a smoke test exercising button/card/input/label/badge/dropdown-menu/sonner.

**Tech Stack:** Next.js 16 (App Router), React 19, Tailwind CSS v4 (CSS-first), shadcn/ui, next-themes, sonner, Radix UI, lucide-react. Package manager: Yarn 1.22.

**Verification note (TDD adaptation):** This is config/UI-setup work with no unit-testable logic. The verification gate for each task is `npx tsc --noEmit` (typecheck), `yarn build` (catches RSC / `"use client"` boundary errors), and a final live dev smoke test — the meaningful equivalents of "make the test pass" for this kind of work.

## Task 4: Build the theme toggle

**Files:**

- [ ] **Step 1: Create the toggle**

Create `components/theme-toggle.tsx`:

```tsx
"use client";

import * as React from "react";
import { Moon, Sun } from "lucide-react";
import { useTheme } from "next-themes";

import { Button } from "@/components/ui/button";
import {
  DropdownMenu,
  DropdownMenuContent,
  DropdownMenuItem,
  DropdownMenuTrigger,
} from "@/components/ui/dropdown-menu";

export function ThemeToggle() {
  const { setTheme } = useTheme();

  return (
    <DropdownMenu>
      <DropdownMenuTrigger asChild>
        <Button variant="outline" size="icon">
          <Sun className="h-[1.2rem] w-[1.2rem] scale-100 rotate-0 transition-all dark:scale-0 dark:-rotate-90" />
          <Moon className="absolute h-[1.2rem] w-[1.2rem] scale-0 rotate-90 transition-all dark:scale-100 dark:rotate-0" />
          <span className="sr-only">Toggle theme</span>
        </Button>
      </DropdownMenuTrigger>
      <DropdownMenuContent align="end">
        <DropdownMenuItem onClick={() => setTheme("light")}>Light</DropdownMenuItem>
        <DropdownMenuItem onClick={() => setTheme("dark")}>Dark</DropdownMenuItem>
        <DropdownMenuItem onClick={() => setTheme("system")}>System</DropdownMenuItem>
      </DropdownMenuContent>
    </DropdownMenu>
  );
}
```
